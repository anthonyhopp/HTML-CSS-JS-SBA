Swanky Doggie Den Website
Welcome to Swanky Doggie Den, the ultimate destination for your furry friend's needs. Our website features four main pages: Main Page, About Us Page, Services Page, and Contact Us Page.

Main Page
The main page serves as a hub for navigating the website. It features links to the other pages, making it easy for users to explore our website. Additionally, each page has a home button that returns users to the main page.

About Us Page
The About Us page tells the story of Swanky Doggie Den. Here, you will learn about our company and what we offer. We are dedicated to providing the best possible care and services for your beloved dogs.

Services Page
The Services Page provides an overview of the services we offer. The page includes a drop-down menu where users can select the services they need and add them to their cart. The page also features a button for submitting service requests.

Contact Us Page
The Contact Us Page provides an easy way for users to get in touch with our company. The page features a form that requires users to input their name, email, and type of inquiry in a drop-down menu. Additionally, there is a message text area where users can submit messages to the company.

Thank you for visiting Swanky Doggie Den's website. We hope you find everything you need to take care of your furry friend. If you have any questions or comments, please feel free to contact us.
